How to Play

  • Roll 3 white dice + 4 color dice
  • Each turn, select 2-3 dice that sum to mark a number
  • All players can mark using white dice combinations
  • Active player can also use white + one color die
  • Mark numbers in order within each row (left→right for Red/Yellow, right→left for Green/Blue)
  • Lock a row when it has 7+ crosses and includes the final number
  • Game ends when 2 rows lock or a player gets 4 penalties
